# 🎮 Game Store Landing Page

A modern and responsive landing page for a **digital game store** built using **JavaScript**, **HTML**, and **CSS**.

---

## 🌟 Features
- 🎨 Sleek and modern UI for showcasing games and offers  
- 🔽 Smooth scrolling navigation for better browsing experience  
- 🕹️ Interactive elements for better user engagement  
- 📱 Fully responsive design for mobile, tablet, and desktop  
- 🛒 Sections for featured games, categories, and deals  

---

## 🛠️ Tech Stack
- JavaScript  
- HTML  
- CSS  
- SCSS (used for styling)
